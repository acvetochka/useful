# Markdown

[Markdown](https://github.com/acvetochka/useful/blob/main/Documentation/Markdown/Markdown.md) - Повний гайд з Markdown, який охоплює все: від базового форматування до складних функцій розширених парсерів (GitHub Flavored Markdown).  
[Markdown-code](https://github.com/acvetochka/useful/blob/main/Documentation/Markdown/Markdown-code.md) - Перелік популярних мов програмування, розмітки, конфігурації та скриптів для підсвічування синтаксису(після ```)  
[Symbols](https://github.com/acvetochka/useful/blob/main/Documentation/Markdown/Symbols.md) - підбірка Unicode-символи та Emoji, розбита за категоріями  
[Mermaid](https://github.com/acvetochka/useful/blob/main/Documentation/Markdown/Mermaid.md) - потужний інструмент, який перетворює текстовий код на графічні схеми.
