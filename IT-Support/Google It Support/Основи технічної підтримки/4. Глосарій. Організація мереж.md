# Нові терміни і їхні визначення: Організація мереж

| Термін                                                                  | Визначення                                                                                                                                                                                                                                                                           |
| ----------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **ARPANET**                                                             | Перша версія Інтернету, створена в межах урядового проєкту DARPA в 1960-х роках. Вона стала основою для подальшого розвитку Інтернету                                                                                                                                                |
| **Закон США "Про захист персональних даних дітей в Інтернеті" (COPPA)** | Закон, що регламентує інформацію, яку ми показуємо дітям віком до 13 років                                                                                                                                                                                                           |
| **Клієнт**                                                              | Пристрій, який отримує дані із сервера                                                                                                                                                                                                                                               |
| **Агентство передових оборонних дослідницьких проєктів (DARPA)**        | Державний проєкт США в 1960-х роках, у межах якого було створено першу версію Інтернету, що стала основою для його подальшого розвитку                                                                                                                                               |
| **Доменне ім’я**                                                        | Назва вебсайту, а саме частина URL-адреси після фрагмента `www`                                                                                                                                                                                                                      |
| **Система доменних імен (DNS)**                                         | Глобальний і широко розподілений мережевий сервіс, що розпізнає IP-адресу за буквеними сегментами (наприклад, назвою вебсайту)                                                                                                                                                       |
| **Кабель Ethernet**                                                     | Кабель, за допомогою якого можна фізично підключитися до мережі                                                                                                                                                                                                                      |
| **Оптоволоконний кабель**                                               | Кабель, що складається з окремих оптичних волокон, тобто тонких скляних трубок завтовшки як людська волосина. На відміну від мідних кабелів, що використовують електричну напругу, волоконні кабелі кодують дані за допомогою світлових імпульсів, які відповідають одиницям і нулям |
| **Глобалізація**                                                        | Рух, завдяки якому уряди, компанії і організації можуть взаємодіяти й співпрацювати на міжнародному рівні                                                                                                                                                                            |
| **Концентратори**                                                       | Пристрої, що слугують центрами, через які передаються дані                                                                                                                                                                                                                           |
| **Інтернет**                                                            | Глобальна система взаємопов’язаних мереж                                                                                                                                                                                                                                             |
| **Корпорація з керування доменними іменами й IP-адресами (ICANN)**      | Організація, де реєструються назви вебсайтів                                                                                                                                                                                                                                         |
| **Інтернет речей**                                                      | Концепція, яка передбачає, що дедалі більше пристроїв під’єднуються до Інтернету в розумніший спосіб. Наприклад, це стосується розумних термостатів, які вимикають або вмикають кондиціонери, коли ви виходите з дому чи повертаєтеся                                                |
| **Internet Protocol версії 4 (IPv4)**                                   | Адреса, що складається з 32 бітів, розділених на чотири групи                                                                                                                                                                                                                        |
| **Internet Protocol версії 6 (IPv6)**                                   | Адреса, яка містить 128 бітів                                                                                                                                                                                                                                                        |
| **Постачальник послуг Інтернету (ISP)**                                 | Компанія, яка надає клієнту інтернет-з’єднання                                                                                                                                                                                                                                       |
| **IP-адреса**                                                           | Найпоширеніший протокол, який використовується на мережевому рівні для спрямування інформації                                                                                                                                                                                        |
| **MAC-адреса**                                                          | Глобальний унікальний ідентифікатор, зв’язаний з окремим мережевим інтерфейсом. Він є 48-бітним номером, який зазвичай складається із шести груп по два шістнадцяткові числа                                                                                                         |
| **Мережа**                                                              | Взаємозв’язок між комп’ютерами                                                                                                                                                                                                                                                       |
| **Перетворення мережевих адрес (NAT)**                                  | Інструмент, який дає організаціям змогу використовувати одну загальнодоступну IP-адресу й багато приватних IP-адрес у мережі                                                                                                                                                         |
| **Організація мереж**                                                   | Проєктування, побудова мереж і керування ними                                                                                                                                                                                                                                        |
| **Мережеві протоколи**                                                  | Набір правил щодо передавання даних у мережі                                                                                                                                                                                                                                         |
| **Мережевий стек**                                                      | Набір апаратного або програмного забезпечення, що забезпечує інфраструктуру для комп’ютера                                                                                                                                                                                           |
| **Маршрутизатор**                                                       | Пристрій для пересилання даних між незалежними мережами                                                                                                                                                                                                                              |
| **Журнали сервера**                                                     | Текстові файли, які містять записану інформацію про дії на певному вебсервері за визначений період часу                                                                                                                                                                              |
| **Сервери**                                                             | Пристрої, що надсилають дані на інші пристрої (клієнти), які запитують ці дані                                                                                                                                                                                                       |
| **Перемикачі**                                                          | Пристрої, які допомагають здійснювати обмін даними                                                                                                                                                                                                                                   |
| **Протокол керування передаванням (TCP)**                               | Протокол, який забезпечує надійне передавання інформації з однієї мережі в іншу                                                                                                                                                                                                      |
| **Уніфікований локатор ресурсів (URL)**                                 | Вебадреса, схожа на домашню адресу                                                                                                                                                                                                                                                   |
| **Атака WannaCry**                                                      | Кібератака, що почалася в Європі й заразила сотні тисяч комп’ютерів у всьому світі                                                                                                                                                                                                   |
| **Бездротові мережі (Wi-Fi)**                                           | Мережі, до яких ви підключаєтеся через радіозв’язок і антени                                                                                                                                                                                                                         |
| **Всесвітня павутина (WWW)**                                            | Інформаційна система, яка надає доступ до документів і інших вебресурсів через Інтернет                                                                                                                                                                                              |
