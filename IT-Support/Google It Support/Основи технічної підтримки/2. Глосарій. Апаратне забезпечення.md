# Нові терміни і їхні визначення: Апаратне забезпечення

| Термін                                              | Визначення                                                                                                                                                                                    |
| --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Адресна шина**                                    | Шина, яка з’єднує ЦП з MCC і надсилає відомості про місцеположення даних, але не самі дані                                                                                                    |
| **ATA**                                             | Найпоширеніший інтерфейс, що використовується для підключення жорстких дисків до системи                                                                                                      |
| **Формфактор ATX (eXtended)**                       | Найпоширеніший формфактор для материнських плат                                                                                                                                               |
| **Зворотна сумісність**                             | Можливість старішого апаратного забезпечення працювати зі своїми новішими версіями                                                                                                            |
| **BIOS (базова система введення/виведення)**        | Програмне забезпечення, яке допомагає ініціалізувати обладнання комп’ютера й запустити операційну систему                                                                                     |
| **Використання власних пристроїв на роботі (BYOD)** | Практика, згідно з якою працівникам дозволяється використовувати власні пристрої в роботі                                                                                                     |
| **Кеш**                                             | Місце, виділене для зберігання останніх або часто використовуваних даних. У мобільному додатку це місце, де зберігається інформація, яку було змінено або створено за допомогою цього додатка |
| **Цикл заряджання**                                 | Одне повне заряджання й розрядження батареї                                                                                                                                                   |
| **Чипсет**                                          | Набір мікросхем, який керує взаємодією компонентів комп’ютера                                                                                                                                 |
| **Тактовий цикл**                                   | Подання напруги на тактовий вхід                                                                                                                                                              |
| **Тактова частота**                                 | Максимальна кількість тактових циклів, що проходять за певний період часу                                                                                                                     |
| **Тактовий вхід**                                   | Компонент ЦП, який отримує напругу під час передавання або отримання даних і повідомляє процесор, що той може почати виконувати обчислення                                                    |
| **ЦП**                                              | Центральний процесор                                                                                                                                                                          |
| **Роз’єми ЦП**                                      | Низки слотів, що з’єднують ЦП і материнську плату комп’ютера                                                                                                                                  |
| **Обсяги даних**                                    | Показники, що стосуються розміру даних, зокрема бітів, байтів, кібібайтів і мегабайтів                                                                                                        |
| **DDR SDRAM**                                       | Тип оперативної пам’яті, що швидше працює, використовує менше енергії і має більший обсяг, ніж попередні версії SDRAM                                                                         |
| **Робочий стіл**                                    | Головний екран, на якому можна переходити між файлами, папками й додатками                                                                                                                    |
| **DIMM**                                            | Двосторонній вбудований модуль пам’яті                                                                                                                                                        |
| **DisplayPort**                                     | Порт, який використовується для виведення аудіо й відео                                                                                                                                       |
| **DRAM**                                            | Динамічна оперативна пам’ять                                                                                                                                                                  |
| **Драйвери**                                        | Програми, що містять вказівки, за допомогою яких ЦП розпізнає зовнішні пристрої, наприклад клавіатури, вебкамери й принтери                                                                   |
| **DVI**                                             | Кабелі, які зазвичай передають лише відео                                                                                                                                                     |
| **Електростатичний розряд**                         | Раптовий і тимчасовий потік електричного струму між двома електрично зарядженими об’єктами, що виникає внаслідок контакту, короткого замикання або діелектричного пробою                      |
| **Зовнішня шина даних (EDB)**                       | Ряд дротів, які з’єднують частини комп’ютера                                                                                                                                                  |
| **Скидання налаштувань**                            | Відновлення на пристрої заводських параметрів                                                                                                                                                 |
| **Формфактор**                                      | Стандарт, що визначає фізичний розмір, форму та компонування апаратного забезпечення                                                                                                          |
| **Жорсткий диск**                                   | Компонент довгострокової пам’яті, який зберігає всі дані, наприклад музику, зображення й додатки                                                                                              |
| **Апаратне забезпечення**                           | Зовнішні або внутрішні пристрої і обладнання, які допомагають виконувати основні функції                                                                                                      |
| **HDD (жорсткий диск)**                             | Диск, який використовує обертову пластину й механічний важіль для читання та записування інформації                                                                                           |
| **HDMI**                                            | Тип кабелю, який передає відео й аудіо                                                                                                                                                        |
| **Радіатор**                                        | Пристрій для розсіювання тепла від ЦП                                                                                                                                                         |
| **Система команд**                                  | Список вказівок, які може виконувати ЦП                                                                                                                                                       |
| **ITX (Information Technology eXtended)**           | Формфактор для материнських плат, значно менший, ніж ATX                                                                                                                                      |
| **Матриця контактних ділянок (LGA)**                | Вид роз’єму ЦП над поверхнею материнської плати                                                                                                                                               |
| **Lightning**                                       | Один зі стандартних типів роз’ємів у мобільних пристроях, який забезпечує живлення, передавання даних і роботу дисплея                                                                        |
| **Мбіт/с**                                          | Одиниця швидкості передавання даних (мегабіт на секунду)                                                                                                                                      |
| **Контролер пам’яті (MCC)**                         | Міст між ЦП й оперативною пам’яттю                                                                                                                                                            |
| **Micro DisplayPort**                               | Тип роз’єму, що використовується для передавання відео й аудіо в компактних пристроях                                                                                                         |
| **Micro HDMI**                                      | Тип роз’єму, що використовується для передавання відео й аудіо в компактних пристроях                                                                                                         |
| **Micro USB**                                       | Тип роз’єму для живлення і передавання даних у мобільних пристроях                                                                                                                            |
| **Mini HDMI**                                       | Тип роз’єму для передавання відео й аудіо в портативних пристроях                                                                                                                             |
| **Mini USB**                                        | Тип роз’єму для передавання даних і живлення                                                                                                                                                  |
| **Материнська плата**                               | Основна плата комп’ютера, яка з’єднує всі його компоненти                                                                                                                                     |
| **Північний міст**                                  | Мікросхема, яка з’єднує оперативну пам’ять, відеокарту та інші високошвидкісні компоненти                                                                                                     |
| **NVMe (NVM Express)**                              | Стандарт інтерфейсу, що забезпечує вищу пропускну спроможність даних і кращу ефективність                                                                                                     |
| **Оверклокінг**                                     | Збільшення тактової частоти ЦП для виконання більшої кількості завдань                                                                                                                        |
| **PCI Express**                                     | Комп’ютерна шина Peripheral Component Interconnect Express                                                                                                                                    |
| **Периферійні пристрої**                            | Зовнішні пристрої, які підключаються до комп’ютера для додаткових функцій, як-от миша, клавіатура й монітор                                                                                   |
| **PGA (pin grid array)**                            | Роз’єм ЦП, на якому безпосередньо розміщено контакти                                                                                                                                          |
| **Порти**                                           | Точки підключення, до яких можна під’єднувати пристрої для розширення функцій комп’ютера                                                                                                      |
| **POST (самотестування після подачі живлення)**     | Процес, що визначає, яке апаратне забезпечення встановлено на комп’ютері                                                                                                                      |
| **Джерело живлення**                                | Блок для перетворення електричної енергії з розетки у формат, потрібний для комп’ютера                                                                                                        |
| **Програми**                                        | Основні вказівки для комп’ютера                                                                                                                                                               |
| **RAM**                                             | Оперативна пам’ять                                                                                                                                                                            |
| **Регістри**                                        | Доступне місце для зберігання даних, з якими працює ЦП                                                                                                                                        |
| **Відновлення образу системи**                      | Процедура очищення й повторного встановлення операційної системи за допомогою образу диска, який є її копією                                                                                  |
| **Дозвіл на повернення товарів (RMA)**              | Процедура повернення товару й отримання дозволу на відшкодування                                                                                                                              |
| **Мікросхема ППЗ (постійна пам’ять)**               | Мікросхема пам’яті, де зберігається BIOS                                                                                                                                                      |
| **RPM**                                             | Оберти за хвилину                                                                                                                                                                             |
| **Безпечна температура експлуатації**               | Діапазон температур, у якому потрібно підтримувати акумуляторні батареї, щоб уникнути пошкоджень                                                                                              |
| **SATA**                                            | Популярний стандарт Serial ATA, який використовує один кабель для передавання даних                                                                                                           |
| **SDRAM**                                           | Синхронна динамічна оперативна пам’ять, що працює синхронно з тактовою частотою системи                                                                                                       |
| **SOC (система на чипі)**                           | Система, що поєднує ЦП, оперативну пам’ять і іноді сховище в одній мікросхемі                                                                                                                 |
| **Південний міст**                                  | Мікросхема, що підтримує контролери введення/виведення (наприклад, жорсткі диски та USB-пристрої)                                                                                             |
| **SSD-диск**                                        | Твердотілий накопичувач                                                                                                                                                                       |
| **Розпірки**                                        | Елементи, за допомогою яких піднімають материнську плату для її закріплення в корпусі                                                                                                         |
| **Термопаста**                                      | Речовина, що використовується для кращого передавання тепла між ЦП та радіатором                                                                                                              |
| **Роз’єм Type-C**                                   | Тип конектора USB, який може замінювати багато периферійних з’єднань                                                                                                                          |
| **UEFI**                                            | Уніфікований інтерфейс розширюваної мікропрограми                                                                                                                                             |
| **USB (універсальна послідовна шина)**              | Стандарт для підключення периферійних пристроїв до комп’ютера                                                                                                                                 |
| **USB-C**                                           | Сучасний тип роз’єму USB, який забезпечує живлення, передавання даних і відеосигналу                                                                                                          |
