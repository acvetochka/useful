# Нові терміни і їхні визначення: Програмне забезпечення

| Термін                                  | Визначення                                                                                                                               |
| --------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Прикладне програмне забезпечення**    | Будь-яке програмне забезпечення, створене для виконання певних завдань, наприклад текстовий або графічний редактор, вебпереглядач тощо   |
| **Мова асемблера**                      | Мова, завдяки якій комп’ютерні науковці змогли перетворювати вказівки у форматі, зрозумілому для людей, у код для машин                  |
| **Автоматизація**                       | Засоби для автоматичної роботи процесів                                                                                                  |
| **Кодування**                           | Переклад з однієї мови на іншу                                                                                                           |
| **Компільована мова програмування**     | Мова, у якій використовуються вказівки у форматі, зрозумілому для людей, що потім обробляються компілятором                              |
| **Авторське право**                     | Право, яке застосовується до оригінального твору                                                                                         |
| **.exe**                                | Розширення для виконуваних файлів в ОС Windows                                                                                           |
| **Мікропрограма (firmware)**            | Програмне забезпечення, що постійно зберігається в компоненті комп’ютера                                                                 |
| **Git**                                 | Система керування версіями, яка допомагає відстежувати зміни у файлах і каталогах                                                        |
| **Інтерпретована мова програмування**   | Мова, яка не компілюється заздалегідь, а виконується інтерпретатором під час роботи                                                      |
| **Програмування**                       | Кодування мовою програмування                                                                                                            |
| **Мова програмування**                  | Спеціальна мова, яку використовують розробники, щоб писати вказівки для комп’ютерів                                                      |
| **Скрипт**                              | Коротка програма, що запускається інтерпретатором і перетворює код у вказівки для ЦП під час виконання                                   |
| **Створення скриптів (scripting)**      | Кодування скриптовою мовою                                                                                                               |
| **Програмне забезпечення**              | Нематеріальні інструкції, які вказують апаратному забезпеченню, що потрібно зробити                                                      |
| **Програмна помилка (bug)**             | Помилка програмного забезпечення, що спричиняє неочікувані результати                                                                    |
| **Управління програмним забезпеченням** | Широкий термін, що охоплює будь-яке програмне забезпечення, розроблене для керування проєктами й завданнями або допомоги з цим           |
| **Системне програмне забезпечення**     | Програмне забезпечення, яке відповідає за роботу основної системи. До нього належать, зокрема, інструменти й утиліти операційної системи |
