# Нові терміни і їх визначення: Знайомство з ІТ
| Термін                          | Визначення                                                                                                                                                            |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Абстрагування**               | Розбір і спрощення досить складної системи, щоб її було легше використовувати                                                                                         |
| **Алгоритм**                    | Низка кроків для вирішення конкретної проблеми                                                                                                                        |
| **ASCII**                       | Найдавніший стандарт кодування символів, який містить літери англійського алфавіту, цифри й знаки пунктуації                                                          |
| **Двійкова система**            | Техніка нумерації, яку використовує комп’ютер. Також відома як система числення з основою 2 (base-2)                                                                  |
| **Байт**                        | Одиниця, що містить 8 біт                                                                                                                                             |
| **Кодування символів**          | Система, що використовується для присвоєння двійкових значень символам, щоб їх можна було прочитати                                                                   |
| **Комп’ютер**                   | Пристрій, який зберігає і обробляє дані за допомогою обчислень                                                                                                        |
| **Криптографія**                | Комплексна дисципліна, що вивчає практику кодування й приховування повідомлень від третіх осіб                                                                        |
| **Десяткова система**           | Система числення, що використовує 10 цифр: від нуля до дев’яти                                                                                                        |
| **Цифровий бар’єр**             | Зростання розриву в знаннях і навичках між людьми з цифровою грамотністю й без неї                                                                                    |
| **Інформаційні технології**     | Використання цифрових технологій, зокрема комп’ютерів і Інтернету, для зберігання й обробки даних у корисну інформацію                                                |
| **ОС Linux**                    | Одна з найбільших операційних систем із відкритим кодом, яка широко використовується як в інфраструктурі бізнесу, так і в рішеннях для кінцевих споживачів            |
| **Логічні вентилі**             | Елементи цифрової схеми, що дають змогу транзисторам виконувати складніші завдання, наприклад вирішувати, куди надсилати електричні сигнали залежно від логічних умов |
| **Відкритий код**               | Код у програмному забезпеченні, розробники якого дозволяють іншим розробникам безкоштовно надавати, змінювати й розповсюджувати його                                  |
| **КПК (кишеньковий комп’ютер)** | Мобільний обчислювальний пристрій                                                                                                                                     |
| **Перфокарти**                  | Набір карток з отворами, які призначено для виконання обчислень в автоматичному режимі без введення даних вручну                                                      |
| **Модель RGB**                  | Стандартна модель представлення кольорів (red, green, blue — червоний, зелений, синій)                                                                                |
| **UTF-8**                       | Найпоширеніший стандарт кодування символів на сьогодні                                                                                                                |
