# Нові терміни і їхні визначення: Усунення несправностей

| Термін                                  | Визначення                                                                                         |
| --------------------------------------- | -------------------------------------------------------------------------------------------------- |
| **Помилка 404 ("Не знайдено")**         | Повідомлення про помилку на вебсайтах, яке з’являється, коли сторінку було переміщено або видалено |
| **Повідомлення про помилку**            | Корисний індикатор, який може допомогти вирішити проблему                                          |
| **Відмовлено в дозволі**                | Повідомлення про помилку, що може з’явитися, якщо ви спробуєте відкрити захищений файл             |
| **Основна причина**                     | Основний фактор, який спричиняє низку проблем                                                      |
| **Вирішення проблем (troubleshooting)** | Можливість діагностувати й вирішувати проблеми                                                     |
