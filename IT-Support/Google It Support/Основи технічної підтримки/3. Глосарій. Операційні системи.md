Нові терміни і їхні визначення: Операційні системи

| Термін                                        | Визначення                                                                                                                                                              |
| --------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Android**                                   | Мобільна операційна система на базі Linux                                                                                                                               |
| **Додаток**                                   | Комп’ютерна програма, призначена для певного використання                                                                                                               |
| **BIOS/UEFI**                                 | Низькорівневе програмне забезпечення, яке виконує ініціалізацію апаратних компонентів комп’ютера для забезпечення правильної роботи                                     |
| **Блокова система зберігання**                | Тип сховища, що прискорює обробку даних. У такому сховищі дані зберігаються не в одному довгому фрагменті, а в блоках, завдяки чому до них можна швидше отримати доступ |
| **Запустити (завантажити комп’ютер)**         | Почати роботу комп’ютера                                                                                                                                                |
| **Завантажувач операційної системи**          | Невелика програма, що завантажує операційну систему                                                                                                                     |
| **ChromeOS**                                  | Операційна система на базі Linux, розроблена компанією Google                                                                                                           |
| **Інтерфейс командного рядка (CLI)**          | Оболонка, яка взаємодіє з операційною системою за допомогою текстових команд                                                                                            |
| **Комп’ютерний файл**                         | Об’єкт, що містить дані. У файлі можна зберігати практично будь-що, наприклад документи, зображення, пісні тощо                                                         |
| **Блоки даних**                               | Дані, які можна розбивати на багато фрагментів і записувати на різні частини носія                                                                                      |
| **Дистрибутиви Linux**                        | До поширених дистрибутивів ОС Linux належать Ubuntu, Debian і Red Hat                                                                                                   |
| **Etcher.io**                                 | Інструмент, за допомогою якого можна записати образ встановлення на USB-пристрій і зробити його завантажувальним                                                        |
| **Розширення файлу**                          | Частина назви файлу, що додається в кінці й указує його тип у певних операційних системах                                                                               |
| **Обробка файлів**                            | Процес зберігання даних за допомогою програми                                                                                                                           |
| **Файлова система**                           | Система керування файлами                                                                                                                                               |
| **Finder**                                    | Диспетчер файлів для пристроїв Mac                                                                                                                                      |
| **Папки (каталоги)**                          | Елементи, що використовуються для впорядкування файлів                                                                                                                  |
| **Дефіцит апаратних ресурсів**                | Нестача системних ресурсів, наприклад пам’яті або місця на диску                                                                                                        |
| **HFS+/APFS**                                 | Файлові системи, розроблені компанією Apple Inc.; APFS використовує більше шифрування, ніж HFS+                                                                         |
| **Ім’я хоста**                                | Унікальна назва, що використовується для ідентифікації комп’ютера в мережі                                                                                              |
| **Пристрій введення/виведення (I/O)**         | Пристрій, який виконує введення й виведення інформації (монітори, клавіатури, миші, диски, вебкамери тощо)                                                              |
| **Образ встановлення**                        | Образ операційної системи, який можна використати для її встановлення на пристрій                                                                                       |
| **Керування введенням/виведенням**            | Усе, що може забезпечити вхідні дані або використовується для виведення даних                                                                                           |
| **iOS**                                       | Мобільна операційна система, розроблена компанією Apple Inc.                                                                                                            |
| **Ядро**                                      | Ключовий компонент операційної системи, який створює процеси, планує їх виконання і керує їх завершенням                                                                |
| **Журнали (logs)**                            | Файли, які записують системні події на комп’ютері                                                                                                                       |
| **macOS**                                     | Операційна система від компанії Apple                                                                                                                                   |
| **Керування пам’яттю**                        | Функція ядра, що оптимізує використання пам’яті й забезпечує додатки необхідною оперативною пам’яттю                                                                    |
| **Метадані**                                  | Дані про файл, зокрема автор, тип, дата зміни та права доступу                                                                                                          |
| **Клієнт сервісів термінала Microsoft**       | Програма для створення RDP-з’єднань із віддаленими комп’ютерами                                                                                                         |
| **OpenSSH**                                   | Популярна програма для використання SSH у Linux                                                                                                                         |
| **Операційна система**                        | Комплекс програм, який керує ресурсами комп’ютера й забезпечує взаємодію з ним                                                                                          |
| **ПК (персональний комп’ютер)**               | Комп’ютер, призначений для використання однією людиною                                                                                                                  |
| **Plink (PuTTY Link)**                        | Інструмент командного рядка для створення SSH-з’єднань                                                                                                                  |
| **PowerShell**                                | Оболонка для Windows, яка інтерпретує текстові команди                                                                                                                  |
| **Досвідчений користувач**                    | Користувач комп’ютера, знання якого вищі за середні                                                                                                                     |
| **Керування процесами**                       | Здатність керувати програмами: їх запуском, виконанням, ресурсами та завершенням                                                                                        |
| **Qwiklabs**                                  | Онлайн-платформа для навчання роботі з хмарними сервісами                                                                                                               |
| **Віддалене з’єднання**                       | Можливість підключення до комп’ютера або мережі на відстані                                                                                                             |
| **RDP (протокол віддаленого робочого стола)** | Протокол Microsoft для захищеного віддаленого підключення                                                                                                               |
| **Масштабованість**                           | Здатність системи змінювати продуктивність і витрати залежно від навантаження                                                                                           |
| **Оболонка (shell)**                          | Програма, яка інтерпретує команди й передає їх операційній системі                                                                                                      |
| **SSH (Secure Shell)**                        | Протокол для безпечного віддаленого доступу між комп’ютерами                                                                                                            |
| **Ключ автентифікації SSH**                   | Безпечний метод автентифікації для доступу до іншого комп’ютера                                                                                                         |
| **SSH-клієнт**                                | Програма для створення SSH-з’єднання                                                                                                                                    |
| **SSH-сервер**                                | Програмне забезпечення, яке дозволяє приймати SSH-з’єднання                                                                                                             |
| **Стандартизація**                            | Систематичний спосіб присвоєння назв (наприклад, хостам)                                                                                                                |
| **Swap-простір**                              | Місце на диску, яке використовується як віртуальна пам’ять при нестачі RAM                                                                                              |
| **Система**                                   | Група апаратних і програмних компонентів, що працюють разом                                                                                                             |
| **Системні налаштування**                     | Параметри системи: екран, користувачі, мережа, пристрої тощо                                                                                                            |
| **Панель завдань**                            | Панель із швидким доступом до функцій, інформації про систему й сповіщень                                                                                               |
| **Термінал**                                  | Текстовий інтерфейс для взаємодії з комп’ютером                                                                                                                         |
| **Інтервал часу (time slice)**                | Дуже короткий проміжок часу, виділений процесу для виконання на ЦП                                                                                                      |
| **Ubuntu**                                    | Популярний користувацький дистрибутив Linux                                                                                                                             |
| **Ім’я користувача**                          | Унікальний ідентифікатор облікового запису                                                                                                                              |
| **Користувацький простір**                    | Частина ОС, з якою взаємодіє користувач (додатки, інтерфейси тощо)                                                                                                      |
| **VirtualBox**                                | Додаток для запуску операційних систем у віртуальному середовищі                                                                                                        |
| **Віртуальна машина**                         | Програма, що дозволяє запускати кілька ОС, використовуючи ресурси одного комп’ютера                                                                                     |
| **Віртуальна пам’ять**                        | Поєднання RAM і дискового простору для роботи процесів                                                                                                                  |
| **VPN (віртуальна приватна мережа)**          | Безпечний спосіб підключення до приватної мережі через Інтернет                                                                                                         |
