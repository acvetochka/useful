# Мережевий рівень (Network Layer)
## Вступ

На канальному рівні (Ethernet) пристрої спілкуються через:
- MAC-адреси

👉 Але це працює добре лише:
- в межах локальної мережі (LAN)

## ⚠️ Проблема MAC-адрес
**📌 Чому цього недостатньо:**
- MAC-адреси:
  - унікальні
  - але ❗ не структуровані

**❗ Основні обмеження:**
1. немає географічної логіки
2. неможливо визначити:
     - де знаходиться пристрій
3. не масштабуються для Інтернету

**🧠 Простими словами:**
> MAC = “випадковий серійний номер”,  
який нічого не каже про місцезнаходження

## 🔗 Обмеження локальної мережі

У LAN:

- комутатори можуть:
  - “вивчити” MAC-адреси
  - швидко передавати дані

👉 Але це працює тільки:
> в межах одного сегмента мережі

**📌 Важливий момент:**

Навіть механізми типу:
- ARP

👉 працюють тільки локально

## 🌍 Рішення: мережевий рівень
**📌 Що потрібно:**

Система, яка:
- масштабується глобально
- дозволяє знайти пристрій у світі

**📡 Рішення:**

→ IP

## 🆔 IP-адреса
**📌 Що це:**

`IP-адреса` — це:
> логічна адреса пристрою в мережі

**💡 На відміну від MAC:**
| MAC             | IP         |
| --------------- | ---------- |
| фізична         | логічна    |
| неструктурована | ієрархічна |
| локальна        | глобальна  |


**🧠 Простими словами:**
> IP = “адреса в Інтернеті”  
(як поштовий адрес)

## 📦 Як це працює разом
**📡 Інкапсуляція**

Дані передаються так:
1. дані → IP
2. IP → додає заголовок
3. IP-датаграма → вкладається в:
    - Ethernet-кадр

**🔗 Ланцюг:**
```
Application → Transport → IP → Ethernet → кабель
```

## 📦 IP-датаграма
**📌 Що це:**
> пакет даних на мережевому рівні

**📡 Містить:**
- IP-адресу джерела
- IP-адресу призначення
- службову інформацію

##🔜 Далі у темі:

Ти навчишся:
- читати IP-адреси
- розуміти структуру IP-заголовка
- бачити, як дані “обгортаються” (encapsulation)

## 🧾 Висновок
- MAC працює тільки локально
- Інтернет потребує:
  - масштабованої адресації

👉 рішення:

- IP + мережевий рівень

## 📌 Головна ідея
> Канальний рівень доставляє дані в межах мережі,  
а мережевий рівень — між мережами по всьому світу
