# Як перевести число в двійкову систему

Є 2 основні способи. Покажу найзрозуміліший для початку.

## ✅ СПОСІБ 1: через “ступені двійки” (найкращий для мереж)
**📌 Ідея:**

ми розкладаємо число на суму:
```
128, 64, 32, 16, 8, 4, 2, 1
```

### 🔢 Приклад: 224
**Крок 1.** Записуємо “ваги бітів”
```
128  64  32  16  8  4  2  1
```

**Крок 2.** Дивимось, що входить в 224
- 128 → ✔ (залишається 96)
- 64 → ✔ (залишається 32)
- 32 → ✔ (залишається 0)
- далі все → ✖

**Крок 3.** Записуємо 1 і 0
```
128 64 32 16 8 4 2 1
 1   1  1  0 0 0 0 0
```

**✅ Результат:**
```
224 = 11100000
```

### 🔢 Ще приклад: 100
**Крок 1:**
```
128 64 32 16 8 4 2 1
```

**Крок 2:**
- 128 → ✖
- 64 → ✔ (залишається 36)
- 32 → ✔ (залишається 4)
- 16 → ✖
- 8 → ✖
- 4 → ✔ (залишається 0)

**Крок 3:**
```
128 64 32 16 8 4 2 1
 0  1  1  0  0 1 0 0
```

**✅ Результат:**
```
100 = 01100100
```

### 🔢 Ще простий: 9
**Крок 1:**
```
128 64 32 16 8 4 2 1
```
**Крок 2:**
```
8 → ✔
1 → ✔
```
**Крок 3:**
```
128 64 32 16 8 4 2 1
 0  0  0  0  1 0 0 1
```
**✅ Результат:**
```
9 = 00001001
```

## 🧠 Дуже важлива логіка

Кожен біт — це “вкл/викл”:
- 1 → беремо число
- 0 → не беремо

## 🔥 Швидкий лайфхак (для subnetting)

Ось найважливіші числа, які треба знати:
| Decimal | Binary   |
| ------- | -------- |
| 128     | 10000000 |
| 192     | 11000000 |
| 224     | 11100000 |
| 240     | 11110000 |
| 248     | 11111000 |
| 252     | 11111100 |

👉 вони постійно використовуються в масках

## 📌 Підсумок

Щоб перевести число в двійкову:
1. береш ваги: 128 64 32 16 8 4 2 1
2. “віднімаєш”
3. ставиш:
   - 1 → якщо входить
   - 0 → якщо ні

## 🧠 Головна ідея
> двійкове число — це просто “які степені двійки входять у число”
