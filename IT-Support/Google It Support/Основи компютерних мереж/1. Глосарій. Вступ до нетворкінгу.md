| Термін                                    | Визначення                                                                                                                         |
| ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Біт**                                   | Найменша одиниця представлення даних, яку комп’ютер може обробляти                                                                 |
| **Протокол граничного шлюзу (BGP)**       | Протокол, за допомогою якого маршрутизатори обмінюються маршрутною інформацією між мережами                                        |
| **Трансляція (broadcast)**                | Тип передачі Ethernet, що надсилається всім пристроям у локальній мережі                                                           |
| **Адреса трансляції**                     | Спеціальна адреса призначення, яка використовується для трансляції; складається з усіх бітів, встановлених у 1 (FF:FF:FF:FF:FF:FF) |
| **Категорії кабелів**                     | Класифікація кабелів за характеристиками; основні типи — мідні та оптоволоконні                                                    |
| **Кабелі**                                | Ізольовані дроти, які з’єднують пристрої та дозволяють передавати дані                                                             |
| **CSMA/CD**                               | Метод доступу до середовища, який визначає, коли канал вільний для передачі та виявляє зіткнення                                   |
| **Клієнт**                                | Пристрій, який отримує дані з сервера                                                                                              |
| **Домен зіткнення**                       | Сегмент мережі, в якому одночасно може передавати дані лише один пристрій                                                          |
| **Комп’ютерні мережі**                    | Система взаємодії комп’ютерів для обміну даними                                                                                    |
| **Категорії мідних кабелів**              | Класи кабелів (наприклад Cat5, Cat5e, Cat6), що визначають швидкість передачі та стійкість до перешкод                             |
| **Перехресні перешкоди (crosstalk)**      | Явище, коли сигнал з одного дроту впливає на інший                                                                                 |
| **CRC (перевірка циклічної надмірності)** | Метод перевірки цілісності даних за допомогою математичного обчислення контрольної суми                                            |
| **Пакет даних**                           | Окремий блок двійкових даних, що передається мережею                                                                               |
| **Канальний рівень (Data Link Layer)**    | Рівень, що визначає спосіб передачі даних між пристроями в межах однієї мережі                                                     |
| **MAC-адреса призначення**                | Апаратна адреса пристрою-одержувача в кадрі Ethernet                                                                               |
| **Дуплексний зв’язок**                    | Тип зв’язку, при якому дані можуть передаватися в обох напрямках                                                                   |
| **Ethernet**                              | Найпоширеніший протокол передачі даних у локальних мережах                                                                         |
| **Кадр Ethernet**                         | Структурований блок даних, що передається мережею                                                                                  |
| **Поле EtherType**                        | Поле в кадрі Ethernet, яке визначає протокол вмісту                                                                                |
| **Оптоволоконний кабель**                 | Кабель, що передає дані за допомогою світлових імпульсів через скляні волокна                                                      |
| **П’ятишарова модель**                    | Модель мережевої взаємодії: фізичний, канальний, мережевий, транспортний і прикладний рівні                                        |
| **Контрольна сума кадру (FCS)**           | 4-байтове значення для перевірки цілісності кадру                                                                                  |
| **Повний дуплекс**                        | Передача даних в обох напрямках одночасно                                                                                          |
| **Напівдуплекс**                          | Передача можлива в обох напрямках, але не одночасно                                                                                |
| **Шістнадцяткова система (hex)**          | Система числення з основою 16                                                                                                      |
| **Хаб**                                   | Пристрій фізичного рівня, який передає дані всім підключеним пристроям                                                             |
| **IP (Internet Protocol)**                | Протокол мережевого рівня для маршрутизації даних                                                                                  |
| **ISP**                                   | Компанія, яка надає доступ до Інтернету                                                                                            |
| **Міжмережа (internetwork)**              | Сукупність взаємопов’язаних мереж                                                                                                  |
| **Лінійне кодування (line encoding)**     | Метод перетворення даних у сигнали для передачі                                                                                    |
| **LAN**                                   | Локальна мережа, що об’єднує пристрої в межах обмеженої території                                                                  |
| **MAC-адреса**                            | Унікальний ідентифікатор мережевого інтерфейсу (48 біт)                                                                            |
| **Модуляція**                             | Метод зміни сигналу для передачі даних                                                                                             |
| **Multicast (багатоадресна передача)**    | Передача даних групі пристроїв                                                                                                     |
| **Мережевий рівень**                      | Рівень, що відповідає за передачу даних між мережами                                                                               |
| **Мережевий порт**                        | Фізичний роз’єм для підключення до мережі                                                                                          |
| **Комутатор (switch)**                    | Пристрій канального рівня, що передає дані лише потрібному пристрою                                                                |
| **Вузол**                                 | Будь-який пристрій у мережі                                                                                                        |
| **Октет**                                 | Група з 8 біт                                                                                                                      |
| **OUI**                                   | Перші три октети MAC-адреси, що визначають виробника                                                                               |
| **Модель OSI**                            | 7-рівнева модель мережевої взаємодії                                                                                               |
| **Патч-панель**                           | Пристрій із багатьма мережевими портами для організації підключень                                                                 |
| **Корисне навантаження (payload)**        | Дані без службової інформації                                                                                                      |
| **Фізичний рівень**                       | Рівень, що описує фізичну передачу сигналів                                                                                        |
| **Преамбула**                             | Початкова частина Ethernet-кадру для синхронізації                                                                                 |
| **Протокол**                              | Набір правил для обміну даними                                                                                                     |
| **Маршрутизатор**                         | Пристрій для передачі даних між мережами                                                                                           |
| **Сервер**                                | Пристрій, який надає дані клієнтам                                                                                                 |
| **Симплексний зв’язок**                   | Передача даних лише в одному напрямку                                                                                              |
| **MAC-адреса джерела**                    | Апаратна адреса пристрою, який надіслав дані                                                                                       |
| **SFD (початковий роздільник кадру)**     | Байт, що сигналізує початок кадру після преамбули                                                                                  |
| **TCP**                                   | Протокол транспортного рівня, що забезпечує надійне з’єднання                                                                      |
| **Транспортний рівень**                   | Рівень, що визначає доставку даних між додатками                                                                                   |
| **Кабель "кручена пара"**                 | Найпоширеніший тип мережевого кабелю з пар скручених проводів                                                                      |
| **Unicast**                               | Передача даних одному конкретному отримувачу                                                                                       |
| **UDP**                                   | Протокол без встановлення з’єднання, без гарантії доставки                                                                         |
| **VLAN**                                  | Технологія створення логічних мереж на одному обладнанні                                                                           |
| **Заголовок VLAN**                        | Додаткове поле в кадрі Ethernet, що містить інформацію про VLAN                                                                    |
