# Терміни та визначення з курсу
## А
| Term                                                      | Description                                                                                                                                                                                                                                                                                                                          |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Artificial intelligence (AI)<br>**Штучний інтелект (AI)** | Computer programs that can complete cognitive tasks typically associated with human intelligence<br>**Комп’ютерні програми, які можуть виконувати когнітивні завдання, зазвичай пов’язані з людським інтелектом**                                                                                                                    |
| AI agent<br>**AI-агент**                                  | an AI-powered tool that can autonomously perform tasks with little human oversight<br>**Інструмент на базі ШІ, який може автономно виконувати завдання з мінімальним людським контролем**                                                                                                                                            |
| AI augmentation<br>**Збільшення штучного інтелекту**      | The process of using AI to improve a work product, whether by making it easier to do or higher in quality<br>**Процес використання ШІ для вдосконалення робочого продукту, будь то шляхом полегшення його виконання або підвищення якості**                                                                                          |
| AI automation<br>**Автоматизація AI**                     | The process of using AI to accomplish tasks, without any action on the user’s part<br>**Процес використання ШІ для виконання завдань без будь-яких дій з боку користувача**                                                                                                                                                          |
| AI model<br>**Модель AI**                                 | A computer program trained on a set of data to recognize patterns and perform specific tasks<br>**Комп’ютерна програма, навчена набору даних для розпізнавання шаблонів та виконання конкретних завдань**                                                                                                                            |
| AI tool<br>**Інструмент AI**                              | AI-powered software that can automate or assist users with a variety of tasks<br>**Програмне забезпечення на основі ШІ, яке може автоматизувати або допомагати користувачам у вирішенні різноманітних завдань**                                                                                                                      |
| AI user<br>**Користувач AI**                              | Someone who leverages AI to complete a personal or professional task<br>**Хтось, хто використовує ШІ для виконання особистого чи професійного завдання**                                                                                                                                                                             |
| Allocative harm<br>**Алокативна шкода**                   | Wrongdoing that occurs when an AI system’s use or behavior withholds opportunities, resources, or information in domains that affect a person’s well-being<br>**Порушення, яке виникає, коли використання або поведінка системи ШІ обмежує доступ до можливостей, ресурсів чи інформації в сферах, що впливають на добробут людини** |

## B

| Term                               | Description                                                                                                                                                                                                                                                        |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Biased data<br>**Упереджені дані** | Data that is incomplete, does not accurately represent populations, or includes preferential treatment for certain individuals or groups<br>**Дані, які є неповними, неточно відображають популяції або включають преференційне ставлення до певних осіб чи груп** |

## C
| Term                                                         | Description                                                                                                                                                                                                     |
| ------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Chain-of-thought prompting<br>**Підказка ланцюга міркувань** | A prompting technique that involves requesting a large language model to explain its reasoning processes<br>**Техніка підказки, яка передбачає запит до великої мовної моделі пояснити свій процес міркування** |
| Cognitive task<br>**Пізнавальне завдання**                   | Any mental activity, such as thinking, understanding, learning, and remembering<br>**Будь-яка розумова діяльність, така як мислення, розуміння, навчання та запам’ятовування**                                  |
| Conversational AI tool<br>**Інструмент розмовного ШІ**       | A generative AI tool that processes text requests and generates text responses<br>**Генеративний інструмент ШІ, який обробляє текстові запити та генерує текстові відповіді**                                   |


## D
| Term                                 | Description                                                                                                                                                                                                                                                                      |
| ------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Data bias<br>**Упередженість даних** | A circumstance in which systemic errors or prejudices lead to unfair or inaccurate information, resulting in biased outputs<br>**Обставина, при якій системні помилки або упередження призводять до несправедливої або неточної інформації, що спричиняє упереджені результати** |
| Deepfakes<br>**Діпфейки**            | AI-generated fake photos or videos of real people saying or doing things that they did not do<br>**Підроблені фото або відео реальних людей, створені ШІ, на яких вони говорять або роблять те, чого насправді не було**                                                         |
| Drift<br>**Дрейф**                   | The decline in an AI model's accuracy in predictions due to changes over time that are not reflected in the training data<br>**Зниження точності моделі ШІ в прогнозах через зміни з часом, які не відображені в навчальних даних**                                              |


## F
| Term                                                     | Description                                                                                                              |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| Few-shot prompting<br>**Підказка з кількома прикладами** | A technique that provides two or more examples in a prompt<br>**техніка, яка надає два або більше прикладів у підказці** |


## G
| Term                                 | Description                                                                                                                                                  |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Generative AI<br>**Генеративний ШІ** | AI that can generate new content, like text, images, or other media<br>**ШІ, який може генерувати новий контент, такий як текст, зображення або інші медіа** |

## H
| Term                                                      | Description                                                                                                                                                                                                     |
| --------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Hallucinations<br>**Галюцинації**                         | AI outputs that are not true<br>**Результати ШІ, які не відповідають дійсності**                                                                                                                                |
| Human-in-the-loop approach<br>**Підхід "людина в циклі"** | A combination of machine and human intelligence to train, use, verify, and refine AI models<br>**Поєднання машинного та людського інтелекту для навчання, використання, перевірки та вдосконалення моделей ШІ** |

## I
| Term                                           | Description                                                                                                                                                                                                                                                                                                                                                |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Interpersonal harm<br>**Міжособистісна шкода** | The use of technology to create a disadvantage to certain people that negatively affects their relationships with others or causes a loss of one's sense of self and agency<br>**Використання технологій для створення несправедливого становища для певних людей, що негативно впливає на їхні стосунки або спричиняє втрату відчуття себе та автономії** |

## K
| Term                                    | Description                                                                                                                                                                                                                                                                     |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Knowledge cutoff<br>**Обмеження знань** | The concept that an AI model is trained at a specific point in time, so it doesn’t have any knowledge of events or information after that date<br>**Концепція, згідно з якою модель ШІ навчається в певний момент часу і не має знань про події чи інформацію після цієї дати** |

## L
| Term                                                        | Description                                                                                                                                                                                                                                                                                                              |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Large language model (LLM)<br>**Велика мовна модель (LLM)** | An AI model that is trained on large amounts of text to identify patterns between words, concepts, and phrases so that it can generate responses to prompts<br>**Модель ШІ, навчена на великих обсягах тексту для виявлення закономірностей між словами, поняттями та фразами з метою генерації відповідей на підказки** |

## M
| Term                                               | Description                                                                                                                                                                                                                         |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Machine learning (ML)<br>**Машинне навчання (ML)** | A subset of AI focused on developing computer programs that can analyze data to make decisions or predictions<br>**Підмножина ШІ, що зосереджена на створенні програм, які аналізують дані для прийняття рішень або прогнозування** |
| Multimodal model<br>**Мультимодальна модель**      | An AI model that can accept and learn from multiple types of input, such as images, video, or audio<br>**Модель ШІ, яка може приймати та навчатися з різних типів даних, таких як зображення, відео або аудіо**                     |

## N
| Term                                  | Description                                                                                                                                  |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Natural language<br>**Природна мова** | The way people talk or write when communicating with each other<br>**Спосіб, у який люди говорять або пишуть під час спілкування між собою** |

## O
| Term                                                 | Description                                                                                                               |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| One-shot prompting<br>**Підказка з одним прикладом** | A technique that provides a single example in a prompt<br>**Техніка, яка надає один приклад у підказці**                  |
| Open dataset<br>**Відкритий набір даних**            | A dataset that is freely available to anyone to use<br>**Набір даних, який є вільно доступним для використання будь-ким** |

## P
| Term                                        | Description                                                                                                                                                                                                                    |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Privacy<br>**Конфіденційність**             | The right for a user to have control over how their personal information and data are collected, stored, and used<br>**Право користувача контролювати, як його персональні дані збираються, зберігаються та використовуються** |
| Prompt<br>**Підказка**                      | Text input that provides instructions to the AI model on how to generate output<br>**Текстове введення, яке надає інструкції моделі ШІ щодо генерації результату**                                                             |
| Prompt engineering<br>**Промпт-інжиніринг** | The practice of developing effective prompts that elicit useful output from generative AI<br>**Практика створення ефективних підказок для отримання корисних результатів від генеративного ШІ**                                |

## Q
| Term                                                       | Description                                                                                                                                                                                                            |
| ---------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Quality-of-service harm<br>**Шкода якості обслуговування** | A circumstance in which AI tools do not perform as well for certain groups of people based on their identity<br>**Ситуація, коли інструменти ШІ працюють гірше для певних груп людей залежно від їхньої ідентичності** |

## R
| Term                                               | Description                                                                                                                                                                                                                            |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Representational harm<br>**Репрезентативна шкода** | An AI tool’s reinforcement of the subordination of social groups based on their identities<br>**Підсилення підпорядкованого становища соціальних груп на основі їхньої ідентичності за допомогою ШІ**                                  |
| Responsible AI<br>**Відповідальний ШІ**            | The principle of developing and using AI ethically, with the intent of benefiting people and society while avoiding harm<br>**Принцип етичної розробки та використання ШІ з метою користі для людей і суспільства та уникнення шкоди** |

## S
| Term                                               | Description                                                                                                                                                                                                                                                                                  |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Security<br>**Безпека**                            | The act of safeguarding personal information and private data, and ensuring that the system is secure by preventing unauthorized access<br>**Захист персональної інформації та приватних даних і забезпечення безпеки системи шляхом запобігання несанкціонованому доступу**                 |
| Social system harm<br>**Шкода соціальній системі** | Macro-level societal effects that amplify existing class, power, or privilege disparities, or cause physical harm, as a result of the development or use of AI tools<br>**Суспільні ефекти на макрорівні, які посилюють існуючі нерівності або спричиняють шкоду внаслідок використання ШІ** |
| Systemic bias<br>**Системна упередженість**        | A tendency upheld by institutions that favors or disadvantages certain outcomes or groups<br>**Тенденція, підтримувана інституціями, яка створює переваги або недоліки для певних результатів чи груп**                                                                                      |

## T
| Term                                 | Description                                                                                                                                                                                                                                                           |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Training set<br>**Навчальний набір** | A collection of data used to teach AI<br>**Набір даних, який використовується для навчання ШІ**                                                                                                                                                                       |
| Transparency<br>**Прозорість**       | The idea that an AI tool should provide insight into how it works, why it made a particular output, and what factors contributed to that output<br>**Ідея, що інструмент ШІ має пояснювати, як він працює, чому видає певний результат і які фактори на це вплинули** |
