# Команда chmod


## Призначення

`chmod` (change mode) — команда Linux для зміни прав доступу до файлів і каталогів.

Вона змінює permission bits файлу:
- read (r)
- write (w)
- execute (x)

та спеціальні біти:
- SUID
- SGID
- sticky bit

## Синтаксис
```bash
chmod [OPTIONS] MODE FILE
```
де
- `MODE` — нові права доступу.

## Модель прав у Linux

Кожен файл має три групи прав:

| категорія  | опис          |
| ---------- | ------------- |
| owner (u)  | власник файлу |
| group (g)  | група файлу   |
| others (o) | всі інші      |

Права:

| право | значення |
| ----- | -------- |
| r     | read     |
| w     | write    |
| x     | execute  |

Приклад:
```
-rwxr-xr--
```
розбивається на:
```
rwx  r-x  r--
```

## Два режими chmod

1️⃣ символьний (symbolic)
2️⃣ числовий (octal)

### Символьний режим

Формат:
```
[who][operator][permissions]
```

- who  
  
  | символ | значення |
  | ------ | -------- |
  | u      | owner    |
  | g      | group    |
  | o      | others   |
  | a      | all      |

- operator  
  
  | символ | значення   |
  | ------ | ---------- |
  | +      | додати     |
  | -      | прибрати   |
  | =      | встановити |

- permissions
  | символ | значення |
  | ------ | -------- |
  | r      | read     |
  | w      | write    |
  | x      | execute  |

**Приклади**

- Додати execute власнику
  ```bash
  chmod u+x script.sh
  ```
- Забрати write у групи
  ```bash
  chmod g-w file.txt
  ```
- Дати read всім
  ```bash
  chmod a+r file.txt
  ```
- Встановити лише read для owner
  ```bash
  chmod u=r file.txt
  ```

### Числовий режим

Кожне право має **число:**

| право | число |
| ----- | ----- |
| r     | 4     |
| w     | 2     |
| x     | 1     |

**Комбінації:**

| право | число |
| ----- | ----- |
| rwx   | 7     |
| rw-   | 6     |
| r-x   | 5     |
| r--   | 4     |

Приклад
```bash
chmod 755 script.sh
```
це:
```
rwxr-xr-x
```

**Найчастіші права**
| число | права     | використання      |
| ----- | --------- | ----------------- |
| 644   | rw-r--r-- | звичайні файли    |
| 755   | rwxr-xr-x | програми          |
| 600   | rw------- | секретні файли    |
| 700   | rwx------ | приватні каталоги |

**Спеціальні біти**
| біт    | число |
| ------ | ----- |
| SUID   | 4     |
| SGID   | 2     |
| Sticky | 1     |

Приклад:
```bash
chmod 4755 program
```

## Основні ключі
| ключ        | значення                |
| ----------- | ----------------------- |
| -R          | рекурсивно              |
| -v          | показувати зміни        |
| -c          | показувати лише змінені |
| --reference | копіювати права         |

Приклад
```bash
chmod -R 755 project/
```

Копіювання прав
```bash
chmod --reference=file1 file2
```

**Важливий нюанс**

Для каталогу:

| право | значення           |
| ----- | ------------------ |
| r     | бачити список      |
| w     | створювати файли   |
| x     | заходити у каталог |
