# ВЕЛИКА таблиця: grep / sed / awk + regex

## 1. Regex — повний reference
| Категорія        | Символ     | Значення         | BRE     | ERE | PCRE |
| ---------------- | ---------- | ---------------- | ------- | --- | ---- |
| Будь-який символ | `.`        | будь-який символ | ✅       | ✅   | ✅    |
| Початок рядка    | `^`        | anchor           | ✅       | ✅   | ✅    |
| Кінець рядка     | `$`        | anchor           | ✅       | ✅   | ✅    |
| Повтор           | `*`        | 0+               | ✅       | ✅   | ✅    |
| Повтор           | `+`        | 1+               | `\+`    | ✅   | ✅    |
| Опційний         | `?`        | 0 або 1          | `\?`    | ✅   | ✅    |
| Діапазон         | `{n,m}`    | кількість        | `\{ \}` | ✅   | ✅    |
| Клас             | `[abc]`    | набір            | ✅       | ✅   | ✅    |
| Заперечення      | `[^abc]`   | не входить       | ✅       | ✅   | ✅    |
| Група            | `( )`      | grouping         | `\(\)`  | ✅   | ✅    |
| OR               | `\| `      | альтернатива     | ❌       | ✅   | ✅    |
| Backref          | `\1`       | повтор групи     | ✅       | ✅   | ✅    |
| Lazy             | `*?`       | мінімум          | ❌       | ❌   | ✅    |
| Lookahead        | `(?=...)`  | вперед           | ❌       | ❌   | ✅    |
| Lookbehind       | `(?<=...)` | назад            | ❌       | ❌   | ✅    |

## 3. grep — всі можливості
📌 Основи
```bash
grep "pattern" file
```
🔧 Опції
| Опція  | Значення     |
| ------ | ------------ |
| `-i`   | ignore case  |
| `-v`   | інверсія     |
| `-r`   | рекурсивно   |
| `-n`   | номер рядка  |
| `-l`   | тільки файли |
| `-c`   | кількість    |
| `-o`   | тільки збіг  |
| `-w`   | ціле слово   |
| `-x`   | весь рядок   |
| `-A n` | після        |
| `-B n` | до           |
| `-C n` | контекст     |

🔹 Типи regex
| Опція     | Тип  |
| --------- | ---- |
| без опцій | BRE  |
| `-E`      | ERE  |
| `-P`      | PCRE |

📌 Приклади
```bash
grep -E 'cat|dog' file
grep -P '\d+' file
grep -r "error" .
```

## 4. sed — всі можливості
📌 Основа
```bash
sed 's/regex/replacement/' file
```
🔧 Команди
| Команда | Значення       |
| ------- | -------------- |
| `s///`  | заміна         |
| `d`     | видалити       |
| `p`     | print          |
| `a\`    | додати після   |
| `i\`    | додати перед   |
| `c\`    | замінити рядок |

🔹 Флаги
| Флаг | Значення      |
| ---- | ------------- |
| `g`  | всі входження |
| `i`  | ignore case   |
| `p`  | print         |

📌 Приклади
```bash
sed 's/foo/bar/g' file
sed '/error/d' file
sed -n '/pattern/p' file
```

## 5. awk — всі можливості
📌 Основна структура
```bash
awk 'pattern { action }' file
```
🔹 Вбудовані змінні
| Змінна   | Значення        |
| -------- | --------------- |
| `$0`     | весь рядок      |
| `$1..$n` | поля            |
| `NF`     | кількість полів |
| `NR`     | номер рядка     |
| `FS`     | роздільник      |

🔹 Приклади
```bash
awk '{print $1}' file
awk -F: '{print $1}' /etc/passwd
awk '/error/ {print $0}' file
```

🔹 Regex в awk
```bash
awk '/pattern/'
awk '$1 ~ /regex/'
```

## 6. Порівняння інструментів
| Інструмент | Для чого           |
| ---------- | ------------------ |
| grep       | пошук              |
| sed        | редагування потоку |
| awk        | аналіз + логіка    |
