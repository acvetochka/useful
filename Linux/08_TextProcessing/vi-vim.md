# vi / vim — повна довідка

## Що це таке

- `vi` — класичний текстовий редактор у Unix/Linux.
- `vim` (Vi IMproved) — його розширена версія з додатковими можливостями.

👉 У більшості сучасних систем vi — це симлінк на vim.

## Основна ідея: режими

`Vim` — це модальний редактор (на відміну від VS Code).

Основні режими:
| Режим                  | Опис                    |
| ---------------------- | ----------------------- |
| **Normal (командний)** | навігація + команди     |
| **Insert**             | введення тексту         |
| **Visual**             | виділення               |
| **Command-line**       | команди типу `:w`, `:q` |

## Перемикання режимів
| Дія                    | Клавіша |
| ---------------------- | ------- |
| Вставка перед курсором | `i`     |
| Вставка після курсора  | `a`     |
| Новий рядок нижче      | `o`     |
| Вийти в Normal         | `Esc`   |
| Візуальний режим       | `v`     |
| Командний рядок        | `:`     |

## Відкриття файлів
```bash
vim file.txt
vim +10 file.txt        # відкрити з 10-го рядка
vim +/pattern file.txt  # знайти pattern
vim file1 file2         # кілька файлів
```

## Вихід і збереження
| Команда        | Опис                         |
| -------------- | ---------------------------- |
| `:q`           | вийти                        |
| `:q!`          | вийти без збереження         |
| `:w`           | зберегти                     |
| `:wq` або `:x` | зберегти і вийти             |
| `ZZ`           | швидкий вихід із збереженням |

## Навігація
🔸 Базова
| Клавіша | Опис   |
| ------- | ------ |
| `h`     | вліво  |
| `l`     | вправо |
| `j`     | вниз   |
| `k`     | вгору  |

🔸 По словах
| Команда | Опис            |
| ------- | --------------- |
| `w`     | вперед до слова |
| `b`     | назад до слова  |
| `e`     | кінець слова    |

🔸 По рядку
| Команда | Опис          |
| ------- | ------------- |
| `0`     | початок рядка |
| `^`     | перший символ |
| `$`     | кінець        |

🔸 По файлу
| Команда | Опис             |
| ------- | ---------------- |
| `gg`    | початок файлу    |
| `G`     | кінець           |
| `:n`    | перейти до рядка |

## Редагування тексту
🔸 Вставка
| Команда | Опис               |
| ------- | ------------------ |
| `i`     | перед курсором     |
| `a`     | після курсора      |
| `I`     | початок рядка      |
| `A`     | кінець рядка       |
| `o`     | новий рядок знизу  |
| `O`     | новий рядок зверху |

🔸 Видалення
| Команда | Опис           |
| ------- | -------------- |
| `x`     | символ         |
| `dd`    | рядок          |
| `dw`    | слово          |
| `d$`    | до кінця рядка |

🔸 Копіювання / вставка
| Команда | Опис            |
| ------- | --------------- |
| `yy`    | копіювати рядок |
| `yw`    | слово           |
| `p`     | вставити після  |
| `P`     | перед           |

🔸 Заміна
| Команда | Опис            |
| ------- | --------------- |
| `r`     | замінити символ |
| `cw`    | змінити слово   |
| `cc`    | змінити рядок   |
| `C`     | до кінця рядка  |

## Undo / Redo
| Команда    | Опис      |
| ---------- | --------- |
| `u`        | скасувати |
| `Ctrl + r` | повернути |

## Пошук
```vim
/pattern     " вперед
?pattern     " назад
```
| Команда | Опис       |
| ------- | ---------- |
| `n`     | наступний  |
| `N`     | попередній |

## Заміна (substitute)
```vim
:s/old/new/        " в рядку
:%s/old/new/g      " у всьому файлі
:%s/old/new/gc     " з підтвердженням
```

## Visual режим
| Команда  | Опис    |
| -------- | ------- |
| `v`      | символи |
| `V`      | рядки   |
| `Ctrl+v` | блок    |


Після виділення:

- `d` — видалити
- `y` — копіювати
- `>` — відступ

## Робота з буферами і файлами
```vim
:e file.txt     " відкрити файл
:bn             " наступний буфер
:bp             " попередній
:bd             " закрити буфер
```

## Вкладки (tabs)
```vim
:tabnew file.txt
:tabnext
:tabprev
```

## Вікна (split)
```vim
:split file.txt     " горизонтально
:vsplit file.txt    " вертикально
```

Навігація:
```
Ctrl + w + h/j/k/l
```

## Макроси
```vim
qa        " запис макросу в регістр a
...       " дії
q         " завершити
@a        " виконати
```

## Повторення команд
| Команда | Опис                 |
| ------- | -------------------- |
| `.`     | повтор останньої дії |
| `3dd`   | виконати 3 рази      |

## Комбінації (дуже важливо)

Vim = оператор + об’єкт

| Приклад | Значення          |
| ------- | ----------------- |
| `dw`    | delete word       |
| `d$`    | delete to end     |
| `ci"`   | змінити в лапках  |
| `di(`   | видалити в дужках |


## Конфігурація (.vimrc)

Файл:
```bash
~/.vimrc
```
Приклад:
```vim
set number
set relativenumber
set tabstop=4
set shiftwidth=4
set expandtab
syntax on
```

## Корисні налаштування
| Команда               | Опис          |
| --------------------- | ------------- |
| `:set number`         | номери рядків |
| `:set relativenumber` | відносні      |
| `:set hlsearch`       | підсвітка     |
| `:set incsearch`      | живий пошук   |

## Різниця vi vs vim
| Функція        | vi  | vim |
| -------------- | --- | --- |
| підсвітка      | ❌   | ✅   |
| undo історія   | ❌   | ✅   |
| плагіни        | ❌   | ✅   |
| автодоповнення | ❌   | ✅   |

## Швидкий старт (мінімум)
```vim
i       → писати
Esc     → вийти
:wq     → зберегти і вийти
dd      → видалити рядок
yy + p  → копіювати / вставити
```

## Лайфхаки
- `.` — найцінніша команда (повтор)
- `ciw` — швидке редагування слова
- `:%s/.../.../g` — масова заміна
- `gg=G` — автоформат

## Коли використовувати

✔ SSH / сервер  
✔ швидке редагування  
✔ конфіг-файли  
✔ DevOps / Linux  

## Коли НЕ варто

❌ великі проекти без IDE  
❌ складна навігація без плагінів

## Висновок

`vim` — це не просто редактор, а “мова редагування тексту”  
Його сила — у комбінаціях і швидкості.
