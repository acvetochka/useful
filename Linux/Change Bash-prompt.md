# Change Bash-prompt

Зміна привітання Bash (prompt) здійснюється через налаштування змінної середовища `PS1`.

Ось кілька способів, від тимчасового до постійного.

## 1. Тимчасова зміна (до перезавантаження терміналу)
Щоб змінити привітання прямо зараз, введіть команду в терміналі:
```bash
export PS1="AAA-> "
```

Ваш промпт миттєво перетвориться на `AAA->`.

## 2. Постійна зміна (назавжди)
Щоб налаштування зберігалися після перезавантаження, потрібно додати команду в файл ~/.bashrc.

Відкрийте файл `~/.bashrc` у редакторі (наприклад, nano):
```bash
nano ~/.bashrc
```

Додайте в кінець файлу рядок:
```bash
PS1="AAA-> "
```

Збережіть файл (Ctrl+O, Enter) та вийдіть (Ctrl+X).

Застосуйте зміни, не закриваючи термінал:
```bash
source ~/.bashrc
```

## Корисні символи для кастомізації

Замість простого тексту, ви можете використовувати спеціальні коди:
- \u — ім'я користувача (username)
- \h — ім'я комп'ютера (hostname)
- \w — поточна робоча директорія (повний шлях)
- \W — поточна директорія (тільки назва папки)
- \d — дата
- \t — час
- \$ — знак $ для користувача або # для root

Приклад (рольове ім'я + папка):
```bash
export PS1="\u@mybox [\w] \$ "
# Результат: user@mybox [/home/user] $
```

## Додавання кольору
Щоб зробити промпт кольоровим, використовуйте ANSI escape-послідовності. Наприклад, для зеленого кольору:

```bash
PS1="\[\033[01;32m\]AAA->\[\033[00m\] "
```

- `\[\033[01;32m\]` — початок зеленого кольору.
- `\[\033[00m\]` — скидання кольору (щоб команди, які ви вводите, не були зеленими).

Кодування кольорів у терміналі виглядає заплутано, бо воно походить від старих стандартів керування відеотерміналами (ANSI).

1. Як розібрати структуру коду

Типовий блок кольору виглядає так: `\[\e[1;32m\]`
- `\[` та `\]` — кажуть bash, що ці символи не мають ширини (щоб текст не "з'їжджав" при переносі рядка).
- `\e[` або `\033[` — початок керуючої послідовності (Escape-символ).
- `1;32` — параметри стилю та кольору.
- `m` — завершення коду кольору.

2. Як вибрати свій колір

Параметри всередині коду розділяються крапкою з комою.

- Стиль тексту:
  - `0` — звичайний.
  - `1` — жирний (bold) або яскравий.
  - `4` — підкреслений.

- Колір тексту (30-37):
  | Код	| Колір	| Код |	Колір|
  |----|------|  -----| -----|
  | 30	| Чорний	| 34 |	Синій|
  | 31	| Червоний	| 35	| Пурпуровий|
  | 32	| Зелений	| 36	| Блакитний |
  | 33	| Жовтий	| 37	| Білий |

- Колір фону (40-47): Ті самі цифри, але починаються з `4` (наприклад, 41 — червоний фон).

Приклад: `\[\e[1;31;40m\]` — жирний червоний текст на чорному фоні.

Обов'язково: Наприкінці кольорової вставки ставте `\[\e[0m\]`, щоб скинути колір на стандартний, інакше весь наступний текст у терміналі буде розфарбований.

## Чи можна змінити формат дати?

Стандартні коди типу `\d` або `\t` мають жорсткий формат. Щоб налаштувати дату "під себе", потрібно використовувати підстановку команди `$(date ...)`.

У PS1 це виглядає так (через одинарні лапки в export):
```bash
export PS1='($(date +"%H:%M:%S")) \u@\h:\w\$ '
```

Популярні ключі для date:
- `%H:%M` — години та хвилини.
- `%d.%m.%y` — день.місяць.рік (наприклад, 22.02.26).
- `%Y-%m-%d` — стандарт ISO (2026-02-22).

Ви можете переглянути повний список форматів дати в офіційній документації GNU Coreutils.
