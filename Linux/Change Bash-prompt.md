# Change Bash-prompt

Зміна привітання Bash (prompt) здійснюється через налаштування змінної середовища `PS1`.

Ось кілька способів, від тимчасового до постійного.

## 1. Тимчасова зміна (до перезавантаження терміналу)
Щоб змінити привітання прямо зараз, введіть команду в терміналі:
```bash
export PS1="AAA-> "
```

Ваш промпт миттєво перетвориться на `AAA->`.

## 2. Постійна зміна (назавжди)
Щоб налаштування зберігалися після перезавантаження, потрібно додати команду в файл ~/.bashrc.

Відкрийте файл `~/.bashrc` у редакторі (наприклад, nano):
```bash
nano ~/.bashrc
```

Додайте в кінець файлу рядок:
```bash
PS1="AAA-> "
```

Збережіть файл (Ctrl+O, Enter) та вийдіть (Ctrl+X).

Застосуйте зміни, не закриваючи термінал:
```bash
source ~/.bashrc
```

## Корисні символи для кастомізації

Замість простого тексту, ви можете використовувати спеціальні коди:
- \u — ім'я користувача (username)
- \h — ім'я комп'ютера (hostname)
- \w — поточна робоча директорія (повний шлях)
- \W — поточна директорія (тільки назва папки)
- \d — дата
- \t — час
- \$ — знак $ для користувача або # для root

Приклад (рольове ім'я + папка):
```bash
export PS1="\u@mybox [\w] \$ "
# Результат: user@mybox [/home/user] $
```

## Додавання кольору
Щоб зробити промпт кольоровим, використовуйте ANSI escape-послідовності. Наприклад, для зеленого кольору:

```bash
PS1="\[\033[01;32m\]AAA->\[\033[00m\] "
```

- `\[\033[01;32m\]` — початок зеленого кольору.
- `\[\033[00m\]` — скидання кольору (щоб команди, які ви вводите, не були зеленими).
