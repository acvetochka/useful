# df

Команда `df` (disk free) використовується для аналізу вільного та використаного простору на змонтованих файлових системах. На відміну від `du`, яка рахує розмір окремих файлів, df бере дані безпосередньо від ядра ОС про стан дискових розділів. 

## Основні ключі та параметри
| Ключ     | Повна назва      | Опис                                                                          |
| -------- | ---------------- | ----------------------------------------------------------------------------- |
| -h       | --human-readable | Вивід у зручному форматі: Gb, Mb, Kb (степінь 1024).                          |
| -H       | --si             | Те саме, що -h, але у десятковій системі (степінь 1000).                      |
| -T       | --print-type     | Додає колонку з типом файлової системи (напр. ext4, xfs, tmpfs).              |
| -i       | --inodes         | Показує стан інодів (кількість доступних номерів файлів) замість блоків.      |
| -a       | --all            | Включає у список технічні та віртуальні файлові системи (pseudo-filesystems). |
| -l       | --local          | Виводить інформацію лише про локальні диски, ігноруючи мережеві (NFS тощо).   |
| -t [тип] | --type=[тип]     | Фільтрує вивід лише для вказаного типу (наприклад, -t ext4).                  |
| -x [тип] | --exclude-type   | Виключає зі списку файлові системи вказаного типу.                            |
|          | --total          | Додає підсумковий рядок зі сумою всіх дисків у кінці.                         |

## Приклади використання
Класичний перегляд усіх дисків:
```bash
df -h
```
Найпопулярніша команда для швидкої перевірки вільного місця.
***

Перевірка конкретної папки або файлу:
```bash
df -h /home
```
Покаже, на якому саме розділі (partition) знаходиться вказаний шлях.
***

Пошук дисків, де закінчилися іноди:
```bash
df -i
```
Буває, що місце на диску є, але неможливо створити файл, бо вичерпано ліміт на кількість файлів (іноди).
***

Приховати "сміття" (тимчасові системи tmpfs):
```bash
df -h -x tmpfs -x devtmpfs
```
Це залишить у списку лише реальні фізичні розділи.
***

Показати типи систем та загальну суму:
```bash
df -Th --total
```
Виведе назви типів систем та додасть рядок total внизу. 

## Важлива відмінність від `du`
Якщо ви видалили великий файл, але програма (наприклад, сервер бази даних) продовжує його використовувати, `du` покаже, що місця стало більше, а `df` продовжуватиме показувати старий обсяг, доки процес не закриє файл або не перезавантажиться.
