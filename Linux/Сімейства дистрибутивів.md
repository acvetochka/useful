# Сімейства дистрибутивів Linux

Світ Linux складається з сотень дистрибутивів, але більшість із них належать до кількох великих «сімейств», що мають спільне коріння, логіку керування та формати пакетів. 

## 1. Сімейство Debian
Найстаріше та найбільше сімейство, відоме своєю консервативністю та акцентом на стабільність. 

- Особливості: Величезні репозиторії (десятки тисяч пакетів), суворий контроль якості, тривалий цикл оновлень.
- Формат пакетів: `.deb`.
- Менеджер: `APT`
- Головні представники:
  - `Debian`: Основа всього сімейства.
  - `Ubuntu`: Найпопулярніший дистрибутив у світі, орієнтований на простоту.
  - `Linux Mint`: Має інтерфейс, схожий на Windows, ідеальний для новачків.
  - `Kali Linux`: Спеціалізована система для тестування на проникнення та кібербезпеки. 

## 2. Сімейство Red Hat (Enterprise)
Орієнтоване на корпоративний сектор, сервери та робочі станції професіоналів. 

- Особливості: Високий рівень безпеки (SELinux), підтримка комерційних стандартів, впровадження інноваційних технологій (наприклад, Wayland, Pipewire).
- Формат пакетів: `.rpm`.
- Менеджер: `DNF` (раніше `YUM`)
- Головні представники:
  - `RHEL` (Red Hat Enterprise Linux): Платний комерційний продукт.
  - `Fedora`: Полігон для випробування нових технологій перед їх потраплянням у RHEL.
  - `Rocky Linux` / `AlmaLinux`: Безкоштовні аналоги RHEL для серверів. 

## 3. Сімейство Arch Linux
Для користувачів, які хочуть повного контролю над своєю системою. 

- Особливості: Rolling Release (система оновлюється постійно, без перевстановлень версій), мінімалізм («зроби сам»), доступ до AUR (Arch User Repository) — найбільшої бази програм.
- Формат пакетів: .pkg.tar.zst.
- Менеджер: `Pacman`
- Головні представники:
  - `Arch Linux`: Встановлюється через консоль, вимагає знань.
  - `Manjaro`: Дружня до користувача версія Arch зі зручним графічним інсталятором. 

## 4. Сімейство SUSE
Має сильні позиції в європейському бізнесі та серверних рішеннях. 

- Особливості: Унікальний інструмент налаштування YaST (графічний центр управління всім залізом та ПЗ).
- Формат пакетів: `.rpm`.
- Менеджер: `Zypper`
- Головні представники:
  - `openSUSE` (Leap/Tumbleweed): Потужна система для розробників та звичайних користувачів.
  - `SLE` (SUSE Linux Enterprise): Комерційне рішення для великого бізнесу. 

## 5. Самостійні та нішеві дистрибутиви
Деякі системи не базуються на жодному з вищезгаданих сімейств:
- `Alpine Linux`: Надлегкий дистрибутив для Docker-контейнерів (`.apk`).
- `Gentoo`: Користувачі самі компілюють кожен пакет з коду для максимальної продуктивності.
- `Slackware`: Один із найстаріших дистрибутивів, що зберігає традиційний підхід UNIX.
- `NixOS`: Використовує декларативний підхід (вся конфігурація системи описується одним файлом).
