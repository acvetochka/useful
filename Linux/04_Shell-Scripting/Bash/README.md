Структура довідки по Bash-скриптінгу
1. Вступ до Bash-скриптів
    - Що таке Bash
    - Для чого потрібні скрипти
    - Як створити та запустити скрипт
    - Shebang (#!)
    - Права доступу (chmod)
2. Змінні та типи даних
    - Оголошення змінних
    - Використання змінних
    - Спеціальні змінні ($0, $1, $?, $$)
    - Оточення (environment variables)
    - Readonly змінні
3. Ввід та вивід
    - echo, printf
    - read (ввід користувача)
    - Перенаправлення (>, >>, <)
    - Pipes (|)
4. Умови
    - if / elif / else
    - test, [ ], [[ ]]
    - Логічні оператори
    - Порівняння чисел та рядків
5. Цикли
    - for
    - while
    - until
    - break, continue
6. Функції
    - Оголошення функцій
    - Передача аргументів
    - return vs echo
7. Робота з файлами
    - Перевірки (-f, -d, -e)
    - Читання файлів
    - Запис у файли
8. Масиви
    - Індексовані масиви
    - Асоціативні масиви
    - Ітерація
9. Розширення (expansions)
    - Parameter expansion
    - Command substitution
    - Arithmetic expansion
    - Brace expansion
10. Обробка помилок
    - Exit codes
    - set -e, set -u
    - trap
11. Підпроцеси та job control
    - Фонові процеси (&)
    - wait
    - subshell ()
12. Практичні патерни
    - Логування
    - CLI-утиліти
    - Парсинг аргументів (getopts)
