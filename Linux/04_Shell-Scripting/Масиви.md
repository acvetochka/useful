# Масиви

Масиви в Bash — це потужний інструмент для зберігання та обробки списків значень.

## Оголошення та наповнення масиву
Створити масив можна двома основними способами:
- Одразу списком: 
  ```bash
  my_array=(apple banana cherry)
  ```
- Поелементно (додавання або заміна):
  ```bash
  my_array="apple"
  my_array="banana"
  my_array="cherry"
  ```
## Звернення до елементів масиву
Для отримання даних використовуються такі конструкції:
- `${my_array}` — звернення до конкретного елемента за індексом.
- `${my_array[@]}` — звернення до всіх елементів як до окремих об'єктів.
- `${my_array[*]}` — звернення до всіх елементів як до одного рядка.
- `${my_array[@]:2:5}` — отримання зрізу (частини масиву) з 2 по 5 елемент.

## Отримання метаданих масиву
Для керування масивами важливо знати їхні параметри:
- `${#my_array[@]}` — повертає загальну кількість елементів (довжину масиву).
- `${!my_array[@]}` — виводить список усіх індексів (ключів) масиву.

## Видалення
Щоб видалити окремий елемент, використовуйте: 
```bash
unset 'my_array[4]'
```
Щоб видалити масив повністю: 
```bash
unset my_array
```

## Масиви та функції
Функції в Bash не можуть приймати масив як прямий аргумент. Для передачі даних необхідно:
Витягти всі дані з масиву та передати їх як окремі аргументи: 
```bash
myfunc ${myarray[*]}
```
Зібрати їх назад у масив всередині функції за допомогою позиційних параметрів: 
```bash
newarray=("$@")
```
