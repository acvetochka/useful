# Оператори для порівняння чисел та рядків

У Bash для порівняння чисел та рядків використовуються різні набори операторів, які зазвичай розміщуються всередині конструкцій `[ ]` або `[[ ]]`.

## Порівняння чисел
Для числових значень використовуються спеціальні прапорці:
- `-eq`: істина, якщо числа рівні.
- `-ne`: істина, якщо числа не рівні.
- `-gt`: істина, якщо перше число більше за друге.
- `-ge`: істина, якщо перше число більше або рівне другому.
- `-lt`: істина, якщо перше число менше за друге.
- `-le`: істина, якщо перше число менше або рівне другому.

## Порівняння рядків
Для роботи з текстовими даними використовуються такі оператори:
- `=` або `==`: перевірка рядків на повну ідентичність.
- `!=`: істина, якщо рядки не ідентичні.
- `<` та `>`: порівняння рядків за алфавітним порядком (менше або більше).
- `-n`: повертає істину, якщо довжина рядка більша за нуль (рядок не порожній).
- `-z`: повертає істину, якщо довжина рядка дорівнює нулю (рядок порожній).

## Конструкції для порівняння
Джерела виділяють два основні способи оформлення умов:
- `[ ]`: POSIX-сумісна форма. При її використанні пробіли навколо операторів є обов'язковими, а змінні рекомендується брати в лапки, щоб уникнути помилок, якщо вони виявляться порожніми.
- `[[ ]]`: сучасна внутрішня конструкція Bash. Вона безпечніша (не потребує лапок для змінних), підтримує логічні оператори (`&&`, `||`, `!`) та патерн-матчинг (наприклад, перевірку на відповідність шаблону `[[ $str == a* ]]`)
