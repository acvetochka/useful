## Функції 
Функція - це фрагмент коду, який виконує одиницю роботи. У прикладах, які ви бачили досі, ви зіткнулися лише з функцією print (), яка виводить повідомлення на екран. Ви будете часто використовувати цю функцію в цьому курсі, щоб перевірити результати вашого коду. Синтаксис функції print () змодельований у наведеному нижче прикладі. 

```python
# Syntax for printing a string of text. Click Run to check the result.

print("Hello world!")
```

## Ключові слова
Ключове слово - це зарезервоване слово в мові програмування, яке виконує певну мету. У своєму першому прикладі Python ви коротко зіткнулися з ключовими словами **for** та **in**. Зверніть увагу, що ключові слова часто відображаються **жирним шрифтом** у цьому курсі. 

У найближчі кілька тижнів ви також дізнаєтеся такі ключові слова:

- Значення: `True`, `False`, `None`
- Умови: `if`, `elif`, `else` 
- Логічні оператори: `and`, `or`, `not` 
- Цикли: `for`, `in`, `while`, `break`, `continue` 
- Функції: `def`, `return` 

Вам не потрібно вивчати весь цей список зараз. Ми зануримося в кожне ключове слово, коли зустрінемося з ними. У Python є додаткові зарезервовані ключові слова. Якщо ви хочете прочитати про них, відвідайте пов'язану статтю «Ключові слова Python» у розділі Ресурси в кінці цього навчального посібника. 

## Арифметичні оператори
Python може обчислювати числа за допомогою звичайних математичних операторів, а також деяких спеціальних операторів: 

- `x + y` -  Оператор додавання `+` повертає суму x плюс y 
 
- `x - y` - Оператор віднімання `-` повертає різницю x мінус y 
 
- `x * y` - Оператор множення `*` повертає добуток x разів y 
 
- `x/y` -  Оператор ділення `/` повертає коефіцієнт x поділений на y 
 
- `x**y` - Оператор експонент `**`  повертає результат підняття x до степені y 
  
- `x**2` - Квадрат - вираз повертає x в квадраті 
 
- `x**3` - Вираз куба повертає x кубовий 
 
- `x**(1/2)` Квадратний корінь (½) або (0,5) Оператор дробового показника повертає квадратний корінь x 
 
- `x//y` Оператор поділу поверне цілу частину цілого ділення x на y 
 
- `x% y` - Оператор модуло повертає решту частини цілого ділення x на y 
 

## Порядок проведення операцій
Порядок операцій обчислюється зліва направо в наступному порядку:
1. **P**арентези (Дужки): (), {}, []
2. **E**кспоненти (Степінь): x ** y
3. **M**ультиплікація (Множення *) та **D**івіжн (Ділення /)
4. **A**ддиція (Додавання +) та **S**убтракція (Віднімання -)
  
Ви можете виявити, що мнемонічний пристрій **PEMDAS** (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction) буде корисним для запам'ятовування цього порядку.
