# Навчальний посібник: Вступ до Python

Цей навчальний посібник містить стислий підсумок того, що ви дізналися в цьому уроці, і слугує керівництвом для майбутньої практичної вікторини.

У цьому розділі ви дізналися, що Python - це мова програмування загального призначення, яка широко використовується для написання сценаріїв та автоматизації, а також для розробки різноманітних додатків. Python сумісна з більшістю операційних систем, включаючи Windows, Linux і Mac OS, і оновлюється кожні кілька років. Python також може працювати на різних машинах, таких як сервери, робочі станції, ПК, мобільні пристрої, IoT тощо.

Python широко використовується в ІТ-сфері, включаючи ІТ-підтримку, системне адміністрування, веб-розробку, машинне навчання, аналітику даних тощо. За допомогою Python можна обчислювати статистику, запускати сайт електронної комерції, обробляти зображення, взаємодіяти з веб-сервісами та виконувати цілу низку інших завдань. Інструкції на Python схожі на англійську мову, що полегшує її вивчення та розуміння у порівнянні з іншими мовами програмування.

**Python** - це

- скриптова мова загального призначення;

- популярна мова, що використовується для кодування різноманітних додатків;

- часто використовуваний інструмент для автоматизації;

- кросплатформенна мова;

- зручна для початківців.

**Python не є**

- мовою сценаріїв, що залежить від платформи / операційної системи;

- мовою сценаріїв на стороні клієнта;

- суто об'єктно-орієнтованою мовою програмування.


## Порівняння коду з Python
У цій програмі ви дізнаєтеся про написання сценаріїв як на Python, так і на Bash. Наступний код ілюструє синтаксичні відмінності між цими двома мовами:

| Вивід на екран у Python                                | Вивід на екран у Bash                                 |
| ------------------------------------------------------ | ----------------------------------------------------- |
| >> print("Hello, how are you?")<br>Hello, how are you? | >> echo Hello, how are you? <br>  Hello, how are you? |


## Ключові терміни
- **Мова сценаріїв, специфічна для платформи / ОС** - мови сценаріїв, специфічні для платформи, такі як PowerShell (для Windows) і Bash (для Linux), використовуються системними адміністраторами на цих платформах.

- **Мова сценаріїв настороні клієнта** - мови сценаріїв на стороні клієнта, такі як JavaScript, використовуються переважно для веб-програмування. Скрипти передаються з веб-сервера в інтернет-браузер кінцевого користувача, а потім виконуються в браузері.

- **Машинна мова** - це комп'ютерна мова найнижчого рівня. Вона безпосередньо спілкується з обчислювальними машинами за допомогою двійкового коду (одиниць і нулів). У двійковому коді одиниця дорівнює електричному імпульсу, а нуль - відсутності електричного імпульсу. Інструкції машинної мови складаються з перекладу таких мов, як Python, у складні патерни одиниць і нулів.

- **Крос-платформна мова** - мова програмування, сумісна з однією або декількома платформами / операційними системами (наприклад, Windows, Linux, Mac, iOS, Android).

- **Об'єктно-орієнтована мова програ мування** - в об'єктно-орієнтованих мовах програмування більшість елементів коду розглядаються як об'єкти з властивостями, що конфігуруються. Наприклад, поле форми - це об'єкт, який можна налаштувати так, щоб він приймав лише дати у форматі мм/дд/рр, і який можна налаштувати на читання з певної бази даних і запис до неї.

- **Інтерпретатор Python** - це програма, яка читає і виконує код Python, перекладаючи код Python в комп'ютерні інструкції.
