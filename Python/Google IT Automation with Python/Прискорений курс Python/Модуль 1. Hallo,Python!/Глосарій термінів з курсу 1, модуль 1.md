# Глосарій термінів з курсу 1, модуль 1

| Термін                                      | Визначення                                                                                                                                                   |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Автоматизація**                           | Процес заміни ручних дій на такі, що відбуваються автоматично                                                                                                |
| **Мова сценаріїв на стороні клієнта**       | Переважно використовується для веб-програмування; скрипти передаються з веб-сервера в інтернет-браузер кінцевого користувача, а потім виконуються в браузері |
| **Редактори коду**                          | Інструменти, що надають функції, включаючи підсвічування синтаксису, автоматичний відступ, перевірку помилок та автозавершення                               |
| **Комп'ютерна програма**                    | Покроковий список інструкцій, яких дотримується комп'ютер для досягнення наміченої мети                                                                      |
| **Функції**                                 | Багаторазово використовуваний блок коду, який виконує певне завдання                                                                                         |
| **Середовище розробки (IDE)**               | Програмне забезпечення, яке надає комплексні засоби для розробки програмного забезпечення                                                                    |
| **Інтерпретатор**                           | Програма, яка читає та виконує код                                                                                                                           |
| **Вхідні дані**                             | Інформація, яку надає програмі кінцевий користувач                                                                                                           |
| **Логічні помилки**                         | Помилки в коді, які перешкоджають його правильному виконанню                                                                                                 |
| **Машинна мова**                            | Комп'ютерна мова найнижчого рівня. Вона безпосередньо взаємодіє з обчислювальними машинами за допомогою двійкового коду (одиниці та нулі)                    |
| **Об'єктно-орієнтована мова програмування** | Мова програмування, в якій більшість елементів коду розглядаються як об'єкти з конфігурованими властивостями                                                 |
| **Вихідні дані**                            | Кінцевий результат завдання, що виконується функцією або комп'ютерною програмою                                                                              |
| **Мова сценаріїв для конкретної платформи** | Мова, яку використовують системні адміністратори на певних платформах                                                                                        |
| **Програмування**                           | Процес написання програми, яка буде поводитися різними способами                                                                                             |
| **Програмний код**                          | Набір письмових інструкцій для комп'ютера, що керуються правилами, з використанням мови програмування                                                        |
| **Мови програмування**                      | Мови з синтаксисом і семантикою для написання комп'ютерних програм                                                                                           |
| **Python**                                  | Мова програмування загального призначення                                                                                                                    |
| **Інтерпретатор Python**                    | Програма, яка читає та виконує код Python шляхом перекладу коду Python в комп'ютерні інструкції                                                              |
| **Скрипт**                                  | Часто використовується для автоматизації певних завдань                                                                                                      |
| **Семантика**                               | Передбачуване значення або ефект висловлювань або наборів слів як у людській, так і в комп'ютерній мовах                                                     |
| **Синтаксис**                               | Правила побудови кожного висловлювання як у людській, так і в комп'ютерній мовах                                                                             |
| **Змінні**                                  | Використовуються для тимчасового зберігання значень у програмному коді                                                                                       |
