# Ресурси Python

## Детальніше про Python
### Використання Python самостійно
Найкращий спосіб вивчити будь-яку мову програмування - практикувати її самостійно якомога більше. Якщо у вас на комп'ютері встановлений Python, ви можете виконати інтерпретатор, запустивши команду python3 (або просто python на Windows), а закрити його можна, набравши exit () або Ctrl-D.

Якщо у вас ще не встановлено Python на вашій машині, це нормально. Ми пояснимо, як встановити його в майбутньому курсі.

### Додаткові ресурси Python
Хоча цей курс дасть вам інформацію про те, як працює Python і як писати скрипти на Python, ви, швидше за все, захочете дізнатися більше про конкретні частини мови. Ось кілька чудових способів допомогти вам знайти додаткову інформацію: 

Прочитайте [офіційну документацію Python](https://docs.python.org/3/).

Шукайте відповіді або задайте питання на 
[Stack Overflow](https://stackoverflow.com/questions). 

Підпишіться на список [розсилки](https://mail.python.org/mailman3/lists/tutor.python.org/) викладачів Python, де ви можете задавати питання та співпрацювати з іншими учнями Python.

Підпишіться на список [розсилки Python-Announce]((https://mail.python.org/mailman3/lists/python-announce-list.python.org/)), щоб читати про останні оновлення мовою.

### Історія Python та поточний стан
Python був випущений майже 30 років тому і має багату історію. Детальніше про це ви можете прочитати на сторінці Вікіпедії 
[Історія Python](https://en.wikipedia.org/wiki/History_of_Python) або в розділі про 
[історію програмного забезпечення](https://docs.python.org/3.0/license.html) з офіційної документації Python.

`Python` - це мова, що швидко розвивається, з новими версіями регулярно випускаються для впровадження нових функцій, підвищення продуктивності, усунення вразливостей безпеки та покращення загальної зручності використання. Хоча ці оновлення стимулюють інновації, іноді вони можуть вимагати коригування існуючого коду. На щастя, Python пропонує інструменти для полегшення процесу міграції. Для цього курсу ми будемо використовувати Python 3.7.
